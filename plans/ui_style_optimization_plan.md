# UI 样式优化计划 - 古典中国风格改造

## 一、需求概述

将 index.js 插件的 UI 界面从现代科技风格改为古典中国风格（参考美化风格.txt），并支持两种颜色主题：
- **墨色主题**（默认暗色）：暖棕色调，古典书卷感
- **素纱主题**（浅色）：米白色调，素雅纸张感

---

## 二、现有代码分析

### 2.1 美化风格.txt 设计特点

| 特性 | 描述 |
|------|------|
| CSS 变量系统 | 使用 `:root` 定义颜色变量，通过 `body.theme-silk` 类切换主题 |
| 字体 | 宋体系列：`'Noto Serif SC','Source Han Serif CN','Songti SC',serif` |
| 背景纹理 | SVG 噪点纹理，营造纸张质感 |
| 边框风格 | 细边框（1px）、小圆角（border-radius: 2px 或 1px） |
| 印章元素 | `.seal` 类：方形印章标签，带边框和特殊颜色 |
| 竖线装饰 | `.item-box::before`：左侧竖线装饰 |
| 动画 | 柔和过渡（transition: all 0.6s ease）、呼吸动画 |
| 滚动条 | 细滚动条（width: 4px），与主题色协调 |

### 2.2 index.js 现有样式结构

| 位置 | 函数/常量 | 行号区间 | 说明 |
|------|-----------|----------|------|
| 窗口系统样式 | [`injectACUWindowStyles()`](index.js:110) | 118-422 | 独立窗口的 CSS 样式 |
| 可视化编辑器样式 | [`VISUALIZER_CSS_ACU`](index.js:27658) | 27658-28700+ | 主编辑界面的 CSS 样式 |

### 2.3 现有风格问题

- 现代科技风格：蓝色渐变、玻璃态效果、大圆角
- 与古典中国风格差异较大
- 不支持主题切换

## 三、修改方案

### 3.1 CSS 变量系统设计

```css
/* 墨色主题（默认） */
:root {
    --bg-color: #24221f;           /* 主背景色 */
    --border-color: #36332e;       /* 边框色 */
    --text-main: #c1b9ad;          /* 主要文字色 */
    --text-dim: #9e978e;           /* 次要文字色 */
    --text-mute: #645e55;          /* 弱化文字色 */
    --accent: #7d4940;             /* 强调色（印章红） */
    --bg-hover: #2a2824;           /* 悬停背景色 */
    --bg-stats: #211f1c;           /* 统计区背景色 */
}

/* 素纱主题（浅色） */
.theme-silk {
    --bg-color: #f4f1eb;
    --border-color: #e0dacb;
    --text-main: #4a453f;
    --text-dim: #6e675e;
    --text-mute: #9e978e;
    --accent: #8a6b5e;
    --bg-hover: #ebe7de;
    --bg-stats: #f9f8f5;
}
```

- 柔和过渡动画

## 六、风险评估

| 风险 | 影响 | 解决方案 |
|------|------|----------|
| CSS 变量冲突 | 可能影响其他元素 | 使用限定作用域 |
| 字体加载失败 | 显示 fallback 字体 | 保持系统字体作为备选 |
| 主题切换不生效 | 用户偏好丢失 | 检查 localStorage 权限 |
